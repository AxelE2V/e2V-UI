## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/eco2veritas-platform.git
cd eco2veritas-platform

# Install dependencies
npm install

# Start development server
npm run dev
```

### Build for Production

```bash
npm run build
```
